XC-WIDTH
x-c-width
XCHAR EXT
 
( xchar -- n )

n is the number of monospace ASCII characters that take the same space to display as the xchar; i.e., xchar width is always an integer multiple of the width of an ASCII char.

Implementation:
: wc, ( n low high -- ) 1+ , , , ;
