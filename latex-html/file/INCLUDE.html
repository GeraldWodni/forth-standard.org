11.6.2.1714
INCLUDE
 
FILE EXT
 
( i * x "name" -- j * x )

Skip leading white space and parse name delimited by a white space character. Push the address and length of the name on the stack and perform the function of INCLUDED.

See:
11.6.1.1718 INCLUDED, A.11.6.2.1714 INCLUDE.

Rationale:
Typical use:
INCLUDE filename
Implementation:
: INCLUDE ( i*x "name" -- j*x )
   PARSE-NAME INCLUDED ;
Testing:
See F.11.6.2.2144.50 REQUIRED.
